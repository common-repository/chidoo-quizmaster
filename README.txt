Various educational quiz apps.

== Description ==

Chidoo Quizmaster is a collection of educational quiz apps. Activate and configure any available app easily in the wordpress backend. Each Quizmaster app provides a single shortcode for integration in a page. Chidoo Quizmaster is extensible, so you can build custom Quizmaster apps. Currently, Chidoo Quizmaster comes with the following default app(s):

* **The Memory by Chidoo Quizmaster** - A memory game for learning music notation.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `chidoo-quizmaster.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the `Quizmaster` menu, activate the desired Quizmaster app(s).
4. Configure any Quizmaster app as needed.
5. Place the specific shortcode provided by the quizmaster app in question on the page where it should appear.

== Frequently Asked Questions ==

= Why activation an deactivation of single apps? =

An app and all its files and assets are only loaded if a) its shordcode is placed on a page AND b) if it is in activated state which is an advantage if there were multiple apps available.
